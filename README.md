# E-Commerce Website

A responsive e-commerce frontend built with React and Vite. Products are fetched from FakeStore API and displayed in a grid layout.

## Features
- Fetch and display products from FakeStore API using `fetch` and `useEffect`
- Product grid layout with image, title, and price
- Responsive design for mobile and desktop
- "Add to Cart" button UI on each product card
- Built with React hooks for state management

## Tech Stack
- **Frontend**: React, JavaScript, CSS
- **Build Tool**: Vite
- **API**: FakeStore API
- **Deployment**: Vercel / Netlify
